Commandes Principales

Découvrez les commandes essentielles pour utiliser ProjectBot

Gestion des Tâches

/task create [titre] [description] Créer une nouvelle tâche
/task assign [id] [@utilisateur] Assigner une tâche à un membre
/task complete [id] Marquer une tâche comme terminée
/task list Afficher toutes les tâches

Gestion des Projets

/project create [nom] Créer un nouveau projet
/project status [id] Voir le statut d'un projet
/project members [id] Gérer les membres du projet
/project deadline [id] [date] Définir une échéance

Rapports & Statistiques

/report weekly Rapport hebdomadaire
/stats user [@utilisateur] Statistiques d'un utilisateur
/progress [projet_id] Progression d'un projet
/export data Exporter les données
